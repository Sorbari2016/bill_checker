# Bill Checker App

## Description 
A simple application to use in checking the electricity and security bills in ID's place. 
The goal of this project is to build a react app as a means to understand how the following works in React:
- Components
- Hooks, especially the useState hook
- JSX syntax
- Keys
- Events & Event handling

## Tools:
- React, Vite
- HTML
- CSS
- JavaScript
